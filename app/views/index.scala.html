SCAPIG

An open-source API Manager

What is it?

It is an out-of-box API Platform which contains all the main components needed to expose your APIs such as an API Gateway, a Developer Hub and an API publisher.

How does it work?

Producers can publish their APIs through the publisher.
Consumers can then view the documentation on the Developer Hub and access the APIs through the Gateway.

What are the main features?

Oauth 2.0 Integration
Rate limiting
Scalable
Pluggable
Versioning
Active/Active architecture

SCALABLE

Scapig has a microservice architecture.
This allows you to selectively increase the capacity of the components which are under more strain.

Scapig data is stored in MongoDB which has great support on replication over multiple datacenters.

PLUGGABLE

Scapig is based on a series of microservices communicating through REST HTTP requests.
It is all open-source so that they can be replaced by bespoke components if needed.
