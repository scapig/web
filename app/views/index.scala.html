What is it?

It is an out-of-box API Platform which contains all the main components needed to expose your APIs such as an API Gateway, a Developer Hub and an API publisher.

How does it work?

Producers can expose their APIs with the Publisher.
Developers can view the API documentation on the Developer Hub.
Consumers can then access the APIs through the Gateway.
