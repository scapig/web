API PUBLISHER - HOWTO

The Scapig Publisher lets you publish your API from a RAML file.

API Definition

The API Definition is derived from the RAML file being published to the scapig-publisher.
The API will contain a number of endpoints which can be either open, application-restricted or user-restricted.
The API context is determined by the prefix of the API request while the version is determined by the Accept header.
curl -XPOST http://localhost:8040/publish -d %40file.raml
