Root Properties

RAML Root Property Description
title API name
description API description
version API version
baseUri The base URI of your API Gateway

(annotations.scopes) The Oauth 2.0 scopes.

documentation It contains all the sections being displayed on the API Documentation
mediaType The default media types to use for request and response bodies
uses Imported libraries for use within the API.
