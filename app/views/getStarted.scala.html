GET STARTED

Prerequisite

Docker is required to run the packaged version of Scapig.

Install

Download the scapig docker file docker-compose.yml and run the following: 
docker swarm init
docker stack deploy --compose-file docker-compose.yml scapig

This will start all the scapig components.
