API GATEWAY - SANDBOX

What is it for?

API Producers can expose a Sandbox implementation so that third-party developers can integrate with the APIs without interfering with Production.

How does it work?

When the consumer utilizes the Sandbox credentials, the request will be routed to /sandbox/request_path.
An example API can be seen there: https://github.com/scapig/api-example
